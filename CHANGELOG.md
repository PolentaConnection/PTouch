# Changelog

All notable changes to PTouch are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0] - 2025-01-15

### Added
- Initial public release
- Update file access and modification timestamps
- Set times from a reference file or explicit timestamp
- Option to change only access or modification time
- Option to skip creating missing files
